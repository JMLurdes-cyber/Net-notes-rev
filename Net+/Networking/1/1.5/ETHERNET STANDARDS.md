Modern Ethernet cables are varied in statistics, such as speed, connectors or maximum length. The [[IEEE|IEEE standard]] used is the one that describes the specifics of the media used.

A common IEEE standard would look like:


| IEEE Standard | Description         | Media                        | Network Speed          |
| ------------- | ------------------- | ---------------------------- | ---------------------- |
| 1000BASE-T    | Gigabit Ethernet    | [[COPPER CABLINGS\|Copper]]  | 1 Gigabit per second   |
| 10GBBASE-T    | 10 Gigabit Ethernet | [[COPPER CABLINGS\| Copper]] | 20 Gigabits per second |
| 1000BASE-SX   | Gigabit Ethernet    | [[FIBER OPTICS\|Fiber]]      | 1 Gigabit per second   |

You can check the names and their specs by following a formula : speed * type * media
- Speed : the speed of the transmission.
- Type : The model of cable used
- Media: The kind of cable used. Usually separated on [[COPPER CABLINGS|copper]] or  [[FIBER OPTICS|fiber]]

For example 1000BASE-T becomes 1000 * BASE * T:
- 1000 = 1 Gigabit
- BASE = Single Frequency
- T = Twisted copper
