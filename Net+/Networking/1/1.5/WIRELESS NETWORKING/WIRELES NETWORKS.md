For a router or a wireless network device to be approved it must fulfill two obligations:
- It must follow the [[IEEE standards]]
- It must have the [[WI-FI ALLIANCE|Wi-Fi Alliance]] trademark

Older standards use names such as 802.11*var*, but newer standards have more marketable names like "Wi-FI 6"

| IEEE Standard | Generation Name | Frequencies     | Maximum theoretical link rate |
| ------------- | --------------- | --------------- | ----------------------------- |
| 802.11a       | -               | 5 GHz           | 6-54 Mb/s                     |
| 802.11b       | -               | 2.4 GHz         | 1-11 Mb/s                     |
| 802.11g       | -               | 2.4 GHz         | 6-54 Mb/s                     |
| 802.11n       | Wi-Fi 4         | 2.4 / 5 GHz     | 72-600 Mb/s                   |
| 802.11ac      | Wi-Fi 5         | 5 GHz           | 433-6933 Mb/s                 |
| 802.11ax      | Wi-Fi 6         | 2.4 / 5 / 6 GHz | 574-9608 Mb/s                 |
| 802.11be      | Wi-Fi 7         | 2.4 / 5 /6 GHz  | 1376 - 46120 Mb/s             |
The three main kinds of Wi-Fi used are : 
- [[4G]], which is the old standard
- [[5G]], the new standard.
- [[SATELLITE NETWORKING|Satellite Networking]], used on remote/harder to reach areas.
