**VIRTUAL PRIVATE CLOUD GATEAWAYS** a.k.a **INTERNET GATEAWAYS** are digital "tunnels" that allow the VPC to connect to the Internet. It does that by translating the IP/information format between the VPC and the internet, since they cannot directly understand each other. Only one IGW can be connected to one VPC.
