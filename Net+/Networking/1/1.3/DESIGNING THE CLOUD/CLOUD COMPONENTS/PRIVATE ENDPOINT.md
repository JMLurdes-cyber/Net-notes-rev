- Private Endpoints enable private connections between a [[VIRTUAL PRIVATE CLOUDS|VPC]] and supported endpoint provider without using an [[VIRTUAL PRIVATE CLOUD GATEAWAY - INTERNET GATEAWAY|IGW]], [[NAT]] device or [[VPN]]. The name will change with the brand:
	- **Amazon Web Services (AWS) VPC Endpoints**
	- **Azure Private Endpoint**
	- **Google Cloud Platform Service connect**
