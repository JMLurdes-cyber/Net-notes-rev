- HTTP a.k.a. **Hypertext Transfer Protocol**
- It is used mainly for web-server communications, like the use of internet through a browser.
- There are two kinds: Plaintext and encrypted.

## Ports used
It uses one of two ports depending on whether it is plaintext or encrypted:
 - It uses [[TCP]] 80 for plaintext communications
 - It uses [[TCP]] 443 for encrypted communications.
