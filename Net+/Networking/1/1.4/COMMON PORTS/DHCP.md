- DHCP, a.k.a. **Domain Host Configuration Protocol**
- Automatically assigns IP's to any device connected to the network.
- Uses two ports to be able to efficiently separate between client and server.
## Port 
- [[UDP]] 67 for *server-side connections*
- [[UDP]] 68 for *client-side connections*
