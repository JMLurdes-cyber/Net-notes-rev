LDAP is also known as **Lightweight Directory Access Protocol**. It provides a database to store the information about the devices connected to the network, how are they connected, which user/s may/tend to access them and their passwords, easing the organization's device management.


- LDAP a.k.a **Lightweight Directory Access Protocol**
- Provides a database that stores information about all devices in a network, alongside:
	- How are they connected
	- Which users are connected
	- Their passwords

### Ports used
- Uses [[TCP]] 389 for **plaintext** communications
- Uses [[TCP]] 636 for **encrypted** communications
