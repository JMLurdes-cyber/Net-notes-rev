- SNMP or **Simple Network Manager Protocol**
- It is used to gather information, events and statistics from all over the network. There are three versions of it:
	- **V1**: 
		- Useful for gathering information about singular events
		- Cannot multitask
		- Plaintext
	- **V2**:
		- Added bulk gathering of events
		- Added other types of data
		- Plaintext
	- **V3:**
		- Could handle even more kinds of data simultaneously
		- Added even more types of data
		- Added Encryption, Authorization and Integrity options
